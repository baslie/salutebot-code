# Паттерны в Code

Обновлено 15 декабря 2023

Паттерны — формальные правила, которые описывают ключевые слова и выражения, содержащиеся в запросе пользователя. С их помощью смартап может сопоставлять запросы пользователя с стейтами, заданными в сценарии.

Паттерны объявляются в сценарии в декларативном [теге `patterns`](../../sa-dsl/tags/declarative-tags.md#patterns) и описываются с помощью [базовых](base-patterns.md) и [расширенных](advanced-patterns.md) элементов.

Также для улучшения распознавания фраз пользователя вы можете использовать [хинты](https://developers.sber.ru/docs/ru/va/chat/voice-interface/speech-recognition/hints).
