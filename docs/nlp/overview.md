# SmartApp Brain

Обновлено 15 декабря 2023

[![](/assets/nlp/overview/Brain.png)
Brain](overview.md)

**SmartApp Brain** — технология определения смысла фразы пользователей смартапов. Включает готовые к использованию модели машинного обучения. Позволяет создавать классификаторы из необработанных логов и управлять обучающими выборками.

Готовые модели используются для работы с [интентами](https://developers.sber.ru/docs/ru/va/about/intents/overview) и извлечения [именованных сущностей](https://developers.sber.ru/docs/ru/va/chat/voice-interface/command-recognition/entities/named-entity) в реплике пользователя: даты и времени, названий мест, имен, адресов электронной почты и других.

Brain используется во всех [инструментах разработки смартапов](https://developers.sber.ru/portal/categories/salute-app-development) от Graph до SmartApp API.

В этом разделе вы найдете информацию как использовать сервис с помощью обращения к REST API.
