# SmartApp DSL

Обновлено 15 декабря 2023

[![](/assets/sa-dsl/overview/Code.png)
Code](../overview.md)

**SmartApp DSL** — язык, предназначенный для разработки логики работы смартапа. SmartApp DSL предоставляет удобный формат для описания конечного автомата, по которому работает смартап.

Файл сценария — древовидная структура. Степень вложенности управляется отступами, подобно языкам `python` и `yaml`.

В SmartApp DSL используются языковые конструкции двух типов:

* **декларативные теги** — описывают структуру смартапа, задают зависимости между файлами, правила перехода между состояниями;
* **теги реакций** — выполняются при попадании диалога в определенное состояние.
