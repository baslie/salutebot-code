# Code

Обновлено 29 декабря 2023

[![](/assets/overview/Code.png)
Code](overview.md)

**Code** — это среда разработки на языках JavaScript и [SmartApp DSL](sa-dsl/overview.md). С помощью Code вы получаете доступ к редактору интентов, сущностей и сценариев, а также к развитым аналитическим инструментам.

Чтобы использовать Code, зарегистрируйтесь в [Studio](https://developers.sber.ru/studio/login) и создайте [проект Code](https://developers.sber.ru/docs/ru/va/chat/script/project-creation#sozdanie-proekta-code).

![Интерфейс Code](/assets/overview/smartapp-code-interface-06a2e047c3cf2bd2e151de3171a4d3f8.png)

Проекты, разрабатываемые в Code, отмечаются в списке приложений ярлыком **Code**.

## Технические требования

Смартапы, созданные в Code, будут работать на всех устройствах, независимо от браузера и типа устройства.

Однако для создания смартапа необходимо соблюсти ряд требований.

### Google Chrome

Code работает только в браузере [Google Chrome](https://www.google.ru/chrome/) . На данный момент мы не тестировали функциональность в других браузерах и не можем гарантировать корректную работу. Поэтому используйте Google Chrome.

### Использование на ПК

Для работы со средой разработки Code необходимо использовать ноутбук или стационарный компьютер.

### Стабильный интернет

Для корректной работы вам потребуется стабильное подключение к сети Интернет.

У браузера Google Chrome, в свою очередь, есть собственные требования. Ознакомиться с ними можно [на этой странице](https://support.google.com/chrome/a/answer/7100626?hl=ru) .
