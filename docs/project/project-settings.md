# Настройка проекта в Code

Обновлено 15 декабря 2023

[![](/assets/project/project-settings/Code.png)
Code](../overview.md)

Чтобы настроить параметры проекта, откройте окно редактирования в **Настройках проекта**.

В окне редактирования проекта доступны:

* Вкладка **Наименование** содержит имя, данное проекту при создании, необязательное описание и информацию о языке проекта. Язык проекта может быть русским или английским. Параметр определяет язык относительно которого проверяются ошибки и классифицируются реплики пользователей.

  Имя и язык проекта изменить нельзя.
* Вкладка **Размещение** содержит параметры внешнего репозитория с кодом проекта.
* Вкладка **Дополнительно** содержит дополнительные параметры проекта. Например, возможность включения автопрослушивания после ответа и запись логов смартапа.
* Вкладка **Экспорт/Импорт** позволяет скачать zip-архив проекта или загрузить в проект файлы с вашего компьютера.

  Перед импортом убедитесь, что структура проекта соответствует [стандартной](overview.md), а все необходимые файлы и папки упакованы непосредственно в архив.
* Вкладки **Классификатор** и **Настройки NLU** содержат параметры работы с инструментом распознавания речи SmartApp Brain. Подробнее параметры описаны в [разделе Настройка проекта раздела SmartApp Brain](../nlp/project/nlp-settings.md)

На вкладке **Классификатор**, в поле **API-ключ Brain**, вы можете получить ключ, необходимый для обращения к [SmartApp Brain Direct API](../nlp/api/overview.md).

Чтобы применить заданные параметры, нажмите **Сохранить**.
