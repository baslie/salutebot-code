# Поиск по файлам проекта Code

Обновлено 25 декабря 2023

[![](/assets/project/file-search/Code.png)
Code](../overview.md)

В редакторе сценариев реализован поиск текста по файлу проекта.

Для поиска:

1. Перейдите в проект смартапа на Code. Нажмите на панели управления **Редактор > Сценарии**. Выберите файл, в котором будет производиться поиск.
2. Нажмите сочетание клавиш `Сtrl+F` или нажмите на иконку поиска внизу страницы.
3. В открывшемся поле введите текст для поиска. Нажмите клавишу `Enter` или кнопку *Enter*.
4. В текущем файле будут выделены совпадения. Вы можете перемещаться по результатам, нажимая клавиши стрелок *Вверх* и *Вниз*.



#### Настройка поисковой выборки

Включите на поле поиска одну или несколько настроек выборки, кликнув на иконки справа:

* чувствительность к регистру;
* поиск по слову;
* использование регулярных выражений.

![поиск по файлу](/assets/project/file-search/base64_2_527fa013.png)


#### Горячие клавиши

Вы можете выделить текст в редакторе, нажать сочетание клавиш `Сtrl+F` для поиска выделенного фрагмента.

* При переходе между файлами поиск не сбрасывается.
* Если вы находитесь в фокусе поля поиска, сбросить результаты можно по нажатию на `esc` (в этом случае поле поиска закроется) или по клику на крестик.
* Вы можете перемещаться по результатам поиска, нажимая клавиши стрелок *Вверх* и *Вниз*.
* Переключайте фокус из поля поиска в редактор по сочетанию клавиш `Сtrl+E` и обратно в поиск по `Сtrl+F`.

Заметили ошибку?

Выделите текст и нажмите `Ctrl` + `Enter`, чтобы сообщить нам о ней
