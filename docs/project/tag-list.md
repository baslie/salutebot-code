# Справочники меток в Code

Обновлено 6 марта 2024

[![](/assets/project/tag-list/Code.png)
Code](../overview.md)

Для перехода к справочнику меток нажмите **Настройки** в левом верхнем углу главной страницы → **Настройки сервисов** → **SmartApp Graph / Code** → вкладка **Справочник меток**.

Здесь вы можете создать группы меток для сообщений или отредактировать группу **Метки для сессий**.

Проставляйте метки для классификации сессий или сообщений.

Для создания группы меток для сообщений:

1. Нажмите **Добавить группу**.
2. Укажите номер группы и название.
3. Нажмите **Создать новую метку** под названием группы.
4. Укажите номер и название метки.
5. Переведите переключатель в активное положение.

Теперь вы можете проставлять метки для сообщений.
