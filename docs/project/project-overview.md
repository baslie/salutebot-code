Страница **Обзор проекта** представляет собой дашборд, на котором отображается аналитическая информация в формате отчетов для текущего проекта.

На дашборде доступны отчеты:

* Уникальные пользователи, сообщения и сессии;
* Перевод на оператора;
* Нераспознанные запросы.

Для переключения между отчетами нажмите на название отчета, например *Уникальные пользователи, сообщения и сессии*, и выберите отчет из выпадающего списка.

Обратите внимание, что данные отчетов отображаются по серверному времени по стандарту `UTC`.

## Уникальные пользователи, сообщения и сессии

Отчет приводит данные по количеству пользователей, сообщений, сессий в зависимости от выбранного периода.

Параметры отчета:

* *Уникальные пользователи* — количество уникальных пользователей за выбранный период.

Под уникальными пользователями подразумевается клиент, который имел одну или более сессий со смартапом за указанный период.

* *Всего сообщений* — количество всех пользовательских сообщений за выбранный период.
* *Всего сессий* — количество всех сессий за выбранный период.

[Подробнее о сессиях](../js-api/session-lifetime-control.md)

> По умолчанию выбранный период составляет последние 7 дней.

Настройка отображения аналитики отчета:

1. Для изменения периода нажмите на иконку календаря и укажите начало и конец временного отрезка.
2. Выберите каналы, для которых будет предоставлена аналитика:

* *Все каналы*;
* *Все каналы, кроме удаленных*;
* *Один или несколько конкретных каналов*.

> По умолчанию в фильтре выбраны *Все каналы*, в том числе и удаленные. Удаленные каналы помечаются в списке как *Удален*.

* При наведении курсора на точки графика отображается количество *Сессий*, *Уникальных пользователей* или *Сообщений* за день. Шаг графика 1 день.
* Нажмите на элемент легенды, чтобы скрыть график.

## Нераспознанные запросы

Отчет представляет количественный анализ нераспознанных запросов пользователей. И позволяет оценить количество нераспознанных запросов в сравнении с общим количеством запросов.

Параметры отчета:

* *Всего запросов* — количество запросов за выбранный период, без учета сообщений после перевода на оператора.
* *Нераспознанных запросов* — количество нераспознанных запросов за выбранный период.
* *Запросы по датам* — график. Отображает количество всех запросов и нераспознанных по дням за выбранный период.
* *Процент нераспознанных запросов* — график. Отображает процент нераспознанных запросов от общего числа за период.

Настройка отображения аналитики:

1. Для изменения периода нажмите на иконку календаря и укажите начало и конец временного отрезка.
2. Выберите каналы, для которых будет предоставлена аналитика:

* *Все каналы*;
* *Все каналы, кроме удаленных*;
* *Один или несколько конкретных каналов*.

> По умолчанию в фильтре выбраны *Все каналы*, в том числе и удаленные. Удаленные каналы помечаются в списке как *Удален*.

* При наведении курсора на точки графика отображается количество *Всех запросов*, *Нераспознанных запросов* или их процентное соотношение за день. Шаг графика 1 день.
* Нажмите на элемент легенды, чтобы скрыть график.

Настройки для проекта являются общими для всех пользователей проекта.
